# teenytinylinks

A free, no-subscription Linktree alternative. All your links live in a single JSON file — no backend, no account, no cost.

**Live demo:** https://thedatacore.github.io/teenytinylinks/

## Get started

```bash
git clone https://github.com/TheDataCore/teenytinylinks.git
cd teenytinylinks
```

Open `links.json` and replace the entries with your own links. Each entry holds a name, URL, and icon type — icons are auto-resolved from the data, no extra config needed.

Once edited, open `index.html` in a browser to preview locally. To publish, push to GitHub and enable **GitHub Pages** from your repo settings (`Settings → Pages → Deploy from main branch`). That's it — your page is live.

## Built with

Vanilla HTML, CSS, and JavaScript. Zero dependencies.
